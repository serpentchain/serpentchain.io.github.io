# An All-Photonic Spectral-Hexadecimal Computing Architecture

**Feasibility, Materials, and Design**

Adam Androulidakis
[serpentchain@gmail.com](mailto:serpentchain@gmail.com) · 1 June 2022 · Conceptual draft

---

## Abstract

Examines the feasibility of an all-photonic computing architecture that performs hexadecimal
logic directly in the optical-wavelength domain, with no intermediary electronic components and
no optical-to-electronic-to-optical conversion.

**Keywords:** optical computing · photonic logic · wavelength encoding · all-optical transistor
· phase-change materials · non-von-Neumann dataflow
